<1조 프로젝트 기능 개요>

1. 콘도 예약/관리 기능 
- 관리자
   - 회원들의 예약 리스트 조회 가능
   - 회원 예약 상세정보 조회, 취소 가능
   - 콘도 이용가격, 이용 가능 여부 수정 가능 

- 회원
   - 비회원은 예약불가
   - 콘도 예약 : 날짜(입실-퇴실) 선택, 객실 선택 후 예약
   - 예약 정보 목록 조회 가능
   - 예약 상세정보 조회 가능
   - 예약 취소 가능


2.  이용권 구매/관리 기능
- 관리자
   - 회원들의 예약 리스트 조회 가능
   - 회원 예약 상세정보 조회, 취소 가능
   - 리프트, 장비(스키, 보드, 의류) 이용가격 수정 가능

- 회원
   - 비회원은 예약불가
   - 스키장 예약 : 날짜 선택, 리프트권 선택 ,장비 사용여부/종류 선택 후 예약 
   - 예약 정보 목록 조회 가능
   - 예약 상세정보 조회 가능
   - 예약 취소 가능
     (장비 - 스키세트(폴, 스키부츠, 스키 플레이트, 보호구), 보드세트(보드부츠, 보드 데크, 보호구), 의류세트(상,하의))


3. 1대1 문의 기능
- 관리자
  - 문의번호, 카테고리, 제목으로 게시글 검색 가능
  - 회원들의 문의 게시글을 조회,  관련없는 문의글 삭제 가능, 문의내용에 답변가능

- 회원
  - 해당 게시판을 이용해 리조트 이용에 관한 문의 게시글을 등록, 삭제, 수정, 조회 가능


4. 후기 게시판 기능
- 관리자
  - 후기 게시판의 글을 관리(조회, 삭제) 할 수 있다. 
  - 관리자는 후기번호, 제목, 등록일자로 게시글 검색 가능
  - 관리자도 게시글에 댓글을 등록, 수정, 삭제, 조회 할 수 있으며, 회원이 남긴 댓글중 부적절한 내용에 대해서 댓글삭제가 가능

- 회원
  - 후기 게시판에 평점과 글 등록(파일첨부 포함), 수정, 삭제, 조회 가능
  - 다른 회원들이 등록한 게시글에 댓글, 대댓글 등록, 조회, 수정, 삭제 가능
  - 후기글에 좋아요 버튼을 누르기 가능


5. 공지사항 기능 (리조트 시설 전체주제)
- 관리자
   - 리조트 이용안내에 대한 공지사항을 등록, 삭제, 수정, 조회 가능
   - 제목 또는 내용으로 공지사항 검색 가능

- 사용자(비회원 포함)
   - 이 게시판을 통해 공지사항을 조회 할 수 있고, 제목 또는 내용으로 공지사항을 검색 가능


6 회원제 시스템 관련 기능
- 공통
  - 로그인, 아이디 찾기, 비밀번호 찾기, 로그아웃 기능 이용 가능

- 관리자
  - 회원 리스트 조회, 회원정보 상세조회, 회원 활동정지 설정/해제 이용 가능
 
- 사용자
  - 회원가입, 내정보 조회 및 마이페이지에서 내 정보 조회 및 수정, 탈퇴 기능 이용 가능


7. 이벤트 게시판 기능
- 공통
   - 이벤트 게시판에서 게시글 목록 조회 가능
   - 게시글을 클릭해서 이벤트 상세조회(제목, 이벤트기간, 이미지, 조회수)로 이동 가능 및 내용 조회 가능

- 관리자
   - 이벤트 게시글을 등록, 수정, 삭제, 조회 가능

- 회원
   - 이벤트 게시글 조회 가능 및 게시글에서 이벤트 응모 가능
   - 당첨자 목록 게시글에서 당첨 여부 확인 가능


8. 스키 강사 채용 기능 
- 관리자
   - 지원자 관리 메뉴
      - 지원서의 제목, 강사명, 지원일자로 지원서 검색 가능
      - 지원서를 pdf파일로 다운로드 가능
      - 지원자들의 서류 정보리스트 조회 가능
      - 3월 1일이 되면 이전까지 제출된 지원서 자동 삭제

   - 강사 모집공고 게시판
      - 강사 모집공고 게시글 등록/조회/수정/삭제 가능
      - 제목을 통해 게시글 검색 가능
      - 3월 1일이 되면 이전까지 모집공고는 자동 삭제

- 사용자(비회원 포함)
   - 모집 공고 글 조회
   - 이름, 주민등록번호 입력후 지원서 작성 가능
   - 이력서 작성 후 '임시 저장' 또는 '제출' 가능
   - 임시 저장된 게시글을 내용 수정, 삭제 가능
   - 제출된 상태일 경우 지원서 수정 및 취소 불가능
