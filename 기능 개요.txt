1조 프로젝트 기능 개요

1. 콘도 , 스키장,패키지(콘도 + 스키장) 예약(결제필수) 기능 

- 관리자
   - 회원들의 예약 리스트 조회 가능
   - 회원 예약 상세정보 조회,취소 가능
   - 콘도 이용가격, 이용 가능 여부 수정 가능 
   - 스키장 이용가격, 이용 가능 여부 수정 가능

- 회원
   - 콘도 예약 : 날짜(입실-퇴실) 선택, 객실 선택 후 예약
   - 스키장 예약 : 날짜 선택,리프트 이용 여부 선택 ,장비 사용여부/종류선택 후 예약 
   - 패키지(리프트 포함) 예약 : 콘도 날짜(입실-퇴실) 선택(스키장,리프트 이용 포함), 객실 선택,장비 사용여부/종류선택 후 예약
   - 예약 정보 목록 조회 가능
   - 예약 상세정보 조회 가능
   - 예약 취소 가능
     (*장비-스키세트(폴,스키부츠,스키,헬멧),보드세트(보드부츠,보드,헬멧))

2. 스키장 장비 관리:

- 관리자
  - 장비들의 종류를 추가, 수정 및 삭제 가능하고 상태(대여가능, 대여불가) 수정 가능

3. 1대1 문의 기능 : (콘도/스키장)

- 관리자
  - 회원들의 문의 게시글을 조회, 삭제 기능. 댓글로 답변가능.(대댓글 불가)
- 회원
  - 해당 게시판을 이용해 리조트 이용에 관한 문의 게시글을 등록, 삭제, 수정, 조회 가능

4. 후기 게시판 기능 (콘도/스키장) 

- 관리자
  - 후기 게시판의 글을 관리(조회, 삭제) 할 수 있다. 
  -관리자도 게시물에 댓글을 등록,수정,삭제,조회 할 수 있으며, 제재가 필요한 댓글삭제가 가능하다.

- 회원
  - 후기 게시판에 평점과 글 등록(파일첨부 포함), 수정, 삭제, 조회 가능. 
  - 다른 회원들이 등록한 게시물에 댓글,대댓글 등록,조회,수정,삭제 가능.
  - 후기글에 좋아요 버튼을 누르기 가능

5. 공지사항 기능 : (리조트전체 공통으로 사용)

- 관리자
   - 리조트 이용안내에 대한 공지사항을 등록, 삭제, 수정, 조회 가능
- 회원
   - 이 게시판을 통해 공지사항을 조회 할 수 있고, 제목 또는 내용으로 공지사항을 검색 가능

6. 분실물 알림게시판 기능 : 

- 관리자
   - 시설에서 습득된 사용자들의 분실물을 알리는 글 등록, 수정, 삭제 조회 가능.
-회원
   - 게시물 조회 가능.

7 회원제 시스템 관련 기능 :

- 관리자
  - 회원목록 조회, 회원정보 상세조회, 회원삭제, 회원정보 수정, 환불 기능 이용 가능.  
- 회원
  - 회원가입, 로그인/로그아웃, 탈퇴, 내정보 조회 및 수정 기능 이용 가능.

8. 픽업서비스 기능 

- 관리자
   - 픽업서비스 가능한 차량의 정보 등록, 수정, 삭제, 조회 가능.
   - 픽업서비스 위치,가격,시간 수정 가능
   - 회원 예약 리스트 조회 가능
   - 회원 예약 상세보기 가능
   - 예약 취소 가능

- 회원
   - 차량선택,위치,날짜,시간 선택 후 예약 가능.
   - 예약한 내역 목록 조회 가능
   - 예약한 내역 상세보기 조회,취소 가능

9.리조트 매출 정보 조회 기능 

- 관리자
   - 리조트의 기간별 매출 정보 조회(파일출력(pdf) 가능)

10. 이벤트 알림 기능

- 관리자
   - 이벤트 알림 게시물을 등록, 수정, 삭제, 조회 가능
- 회원
   - 이벤트 알림 게시물 조회 가능, 제목으로 게시글 검색 가능

11. 스키 강습 예약 기능

- 관리자
   - 클래스 등록(수업날짜,수업시간,수강인원,난이도)수정,삭제 조회 가능.
- 회원
   - 스키장예약 또는 패키지 예약자만 구매 가능.
   - 달력에서 해당 날짜를 선택하면 해당일의 수업 날짜,시간,수강인원,난이도를 선택 후 예약(결제 필수) 가능.  

12. 스키 강사  채용 기능 

- 관리자
   - 강사 모집공고 게시글 등록/수정/삭제 가능
   - 지원자들의 서류 정보리스트 조회 가능,
   - 지원결과 (합격/탈락) 설정 가능.
- 이용자(회원가입 x)
   - 모집 공고 글 조회
   - 이력서 작성,제출 기능
   - 이름,주민등록번호로 지원 내역 목록,합격결과 조회 가능

13.알림함 기능

회원
 - 자신이 쓴 글에 댓글이 달리면 알림이 온다
 - 콘도/스키장/패키지 예약/취소 후 알림을 받는다.
 - 스키 강습 예약/취소 후 알림을 받는다.
 - 내가 쓴 글의 댓글이 등록되면 회원 알림함으로 알림이 전송된다.
 - 새로운 공지사항/이벤트가 등록되면 알림을 받는다.
 - 픽업서비스 예약/취소 후 알림을 받는다
